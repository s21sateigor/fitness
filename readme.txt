Fitness

For Windows:
Dependencies to install:
Select "Install additional tools" in the installation process
https://nodejs.org/en/download
https://spring.io/tools (Eclipse version recommended)

Development environment setup:
	* Make sure to start the Spring Boot App
	* Enter "/app" in bash and launch the command "npm start"
	> Now you have started the back-end and front-end of the app
	* Open the "/app" directory in VSCode, for convenience sake
	* Access the React site by going to http://localhost:3000
